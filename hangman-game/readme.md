# Hangman Game

This is a simple Hangman game built using HTML, CSS, and JavaScript. The game allows users to guess a hidden word by selecting letters. The objective is to guess the word correctly before running out of attempts.

## Table of Contents

- [Demo](#demo)
- [Features](#features)

## Demo

You can play the game [here](https://hangman-practice.netlify.app
).

## Features

- Interactive gameplay with keyboard and on-screen letter selection.
- Tracks the number of attempts left.
- Displays correctly guessed letters and updates the word in real-time.
- Simple and responsive user interface.
